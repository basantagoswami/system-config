# System Setup

This repo contains scripts and dotfiles.

Rational behind installing sway via apt:
- Couldn't install Hyprland and make it work. Couple of weeks later, Ubuntu is getting a new relase which will hopefully have updated packages that we needed. Installing any WM via Nix causes issues with graphics related settings. nixGL or whatever needs to be configured, either manually or using home-manager. It's a pain. Sway via apt is the simplest solution.
